The `AuroraMap` component is a high-performance, WebGL-powered map interface based on **[MapLibre GL JS](https://maplibre.org/maplibre-gl-js/docs/)**. It uses DOM attributes to configure instances automatically without writing a custom initialization script for each map. Maps automatically adapt to Light and Dark mode using open-source Carto Basemaps.

## Installation

MapLibre is loaded on-demand dynamically when a `.aurora-map` or `data-component="map"` element is present on the page.

If you are using this inside a framework like React, you should explore `react-map-gl/maplibre` instead of using this Vanilla JS wrapper.

## Basic Layout

This example initializes a map centered on **New York City** with a default zoom level. Use `data-lat`, `data-lng`, and `data-zoom` to control the initial view state. 

```html
<div class="aurora-map" 
     data-component="map" 
     data-lat="40.7128" 
     data-lng="-74.0060" 
     data-zoom="11">
</div>
```

## Adding Markers and Popups

You can add standard map markers with optional popups by nesting `div.aurora-marker` elements inside the map container. Provide `data-lat`, `data-lng`, and `data-label`.

```html
<div class="aurora-map h-large" data-component="map" data-lat="37.7749" data-lng="-122.4194" data-zoom="12">
    <!-- Golden Gate Bridge -->
    <div class="aurora-marker" 
         data-lat="37.8199" 
         data-lng="-122.4783" 
         data-label="Golden Gate Bridge">
        <span class="material-symbols-outlined">tour</span>
    </div>
    
    <!-- Alcatraz Island -->
    <div class="aurora-marker" 
         data-lat="37.8267" 
         data-lng="-122.4230" 
         data-label="Alcatraz Island">
        <span class="material-symbols-outlined">fort</span>
    </div>

    <!-- Fisherman's Wharf -->
    <div class="aurora-marker" 
         data-lat="37.8080" 
         data-lng="-122.4177" 
         data-label="Fisherman's Wharf">
         <span class="material-symbols-outlined">directions_boat</span>
    </div>
</div>
```

## Geolocation Features

AuroraMap supports configuring free geographical lookups natively via HTML.

### Auto-Locate via IP Address

By adding `data-auto-locate="ip"`, this map will dynamically hit `http://ip-api.com/json` upon initialization and rapidly fly to the viewer's estimated coordinate location, placing an inferred marker.

```html
<div class="aurora-map" 
     data-component="map" 
     data-auto-locate="ip">
</div>
```

<div class="docs-demo">
  <div class="aurora-map" 
       data-component="map" 
       data-auto-locate="ip">
  </div>
</div>

### Browser Native GPS Prompt

By passing `data-controls="geolocate"`, a new crosshair control is injected below the standard zoom actions. When the user clicks this icon, the map launches the browser's native location authorization prompt and streams their GPS directly onto the map as a tracking dot.

```html
<div class="aurora-map" 
     data-component="map" 
     data-lat="51.5072" 
     data-lng="-0.1276" 
     data-zoom="9"
     data-controls="geolocate">
</div>
```

## Map to Data Table Interaction

This complex example demonstrates pulling external CSV data natively to construct a table and attaching click handlers securely via the `map:ready` broadcast generated by `initMaps()`. When you click on a team's row, the map flies dynamically to that location!

<div class="docs-demo">
  <div style="display: flex; gap: 24px;">
    <div style="flex: 1; min-width: 250px; max-width: 300px; display: flex; flex-direction: column;">
      <h4 style="margin-top: 0; margin-bottom: 12px;">English Stadiums</h4>
      <div style="max-height: 400px; overflow-y: auto; border: 1px solid var(--color-border-subtle); border-radius: var(--radius-md);">
        <table class="aurora-table" style="width: 100%; border-collapse: collapse; font-size: var(--text-sm);">
          <thead style="position: sticky; top: 0; background: var(--color-surface-elevated); z-index: 10;">
            <tr>
              <th style="text-align: left; padding: 12px 8px; border-bottom: 1px solid var(--color-border-subtle);">Team</th>
              <th style="text-align: left; padding: 12px 8px; border-bottom: 1px solid var(--color-border-subtle);">Stadium</th>
            </tr>
          </thead>
          <tbody id="stadiums-body">
            <!-- Populated via script -->
          </tbody>
        </table>
      </div>
    </div>
    <div style="flex: 2;">
      <div id="interactive-stadium-map" class="aurora-map" data-component="map" data-lat="53.48" data-lng="-2.2" data-zoom="5"></div>
    </div>
  </div>
</div>

<script>
document.addEventListener('DOMContentLoaded', () => {
    const mapContainer = document.getElementById('interactive-stadium-map');
    const tbody = document.getElementById('stadiums-body');
    
    // Wait for Aurora to lazily initialize the library and broadcast the instance
    mapContainer.addEventListener('map:ready', async (e) => {
        const { map, maplibregl } = e.detail;
        
        try {
            // Fetch the mock CSV mapping
            const res = await fetch('https://raw.githubusercontent.com/jokecamp/FootballData/refs/heads/master/other/stadiums-with-GPS-coordinates.csv');
            const csvText = await res.text();
            
            const rows = csvText.split('\n').filter(r => r.trim() !== '');
            const headers = rows.shift().split(',');

            // We only show the top 10 rows from the CSV to keep this demo fast
            const stadiums = rows.slice(0, 10).map(rowStr => {
                const cols = rowStr.split(',');
                return { team: cols[0], stadium: cols[3], lat: parseFloat(cols[5]), lng: parseFloat(cols[6]) };
            }).filter(s => !isNaN(s.lat) && !isNaN(s.lng));

            // Generate DOM and WebGL markers
            stadiums.forEach(stadium => {
                const marker = new maplibregl.Marker({ color: 'var(--color-brand)' })
                    .setLngLat([stadium.lng, stadium.lat])
                    .setPopup(new maplibregl.Popup({ offset: 25 }).setText(stadium.stadium))
                    .addTo(map);

                const tr = document.createElement('tr');
                tr.style.cursor = 'pointer';
                tr.style.borderBottom = '1px solid var(--color-border-subtle)';
                
                tr.addEventListener('mouseenter', () => tr.style.backgroundColor = 'var(--color-surface-sunken)');
                tr.addEventListener('mouseleave', () => tr.style.backgroundColor = 'transparent');
                
                tr.innerHTML = `<td style="padding: 12px 8px;">${stadium.team}</td><td style="padding: 12px 8px;">${stadium.stadium}</td>`;

                tr.addEventListener('click', () => {
                    map.flyTo({ center: [stadium.lng, stadium.lat], zoom: 15 });
                    marker.togglePopup(); // Optional visual cue
                });

                tbody.appendChild(tr);
            });
        } catch (err) { }
    });
});
</script>

## Design Rules

- **Margins:** Map controls operate 16px from the container edge.
- **Colors:** Popups utilize the system's `var(--color-surface-elevated)` token.
- **Auto-Theming:** The Map automatically shifts between Light (Positron) and Dark (Dark Matter) depending on the global CSS `data-mode` variable.
