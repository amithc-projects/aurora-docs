## Default Blockquote
Standard variant with a primary colored left border.

{{< demo >}}
<blockquote class="blockquote">
  <p>"Flowbite is just awesome. It contains tons of predesigned components and pages starting from login screen to complex dashboard. Perfect choice for your next SaaS application."</p>
  <cite class="blockquote__footer">Flowbite Docs</cite>
</blockquote>
{{< /demo >}}

## Iconic Variant
Uses a stylized quotation mark instead of a border.

{{< demo >}}
<blockquote class="blockquote blockquote--iconic">
  <p>"Aurora's token-driven architecture makes extending the system for new components like this incredibly simple."</p>
  <cite class="blockquote__footer">Aurora Design Team</cite>
</blockquote>
{{< /demo >}}
